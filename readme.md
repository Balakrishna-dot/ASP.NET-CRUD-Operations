\# ASP.NET CRUD Operations



A simple ASP.NET Web Forms application demonstrating CRUD operations

(Create, Read, Update, Delete) using C# and SQLite.



\## Features

\- Create new records

\- View records

\- Update records

\- Delete records

\- SQLite database



\## Technologies Used

\- ASP.NET Web Forms

\- C#

\- SQL Server

\- Visual Studio



\## How to Run

1\. Open the solution in Visual Studio

2\. Restore NuGet packages

3\. Run the project using IIS Express
